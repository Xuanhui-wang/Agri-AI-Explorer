# 农智探客 (Agri-AI Explorer)

> **面向农业院校的人工智能模型可视化学习与仿真实践平台**

## 🌾 项目简介

“农智探客 (Agri-AI Explorer)” 是一款专为农业院校和非信息类专业学生量身打造的人工智能通识课学习工具。本系统完美涵盖了从特征提取到仿真部署的 AI 全流程，旨在解决传统农业 AI 通识课中人工智能模型数学原理推导难懂、缺乏农业实景案例、上机实践环境配置繁琐且缺少直观交互演示环境的四大痛点。

本项目通过 **算法机制白盒化**、**农业实景具象化** 和 **编程环境极简化**，帮助农科学生跨越底层代码与复杂数学推导的障碍，深刻理解深度学习与机器学习的底层逻辑，唤醒学生利用前沿 AI 技术反哺国家第一产业的专业使命感。

---

## ✨ 核心功能模块

系统共包含六大核心教学与实践模块，通过侧边栏可无缝切换：

### 1. 特征透视：病斑提取 (CNN 原理可视化)
* **功能说明**：抛弃抽象的数学推导，直击卷积神经网络 (CNN) 的底层运作原理。
* **使用说明**：
  * 在下拉菜单中选择真实的《跨品种小麦赤霉病(FHB)无人机影像数据集》（支持切换不同发病程度与品种，如豫农912、郑麦1860）。
  * 选择不同的卷积核（边缘检测、锐化、平滑等）。
  * 界面将实时展示 **原始灰度图像矩阵** × **卷积核矩阵** = **特征图** 的点乘过程。
  * 点击“播放矩阵计算动画”，可观看由底层计算驱动的高清滑动卷积动画视频，彻底解构黑盒算法。

### 2. 回归分析：产量预测 (机器学习交互实验室)
* **功能说明**：基于真实水稻产量及农情特征数据，直观感受模型欠拟合与过拟合。
* **使用说明**：
  * 勾选输入特征（降水、温度、光照、施肥）。
  * 拖拽滑块动态调整数据噪声扰动量。
  * 选择算法模型（多元线性回归、随机森林），并可动态调整随机森林的决策树数量。
  * 系统毫秒级刷新回归预测分布图和决定系数 (R²)，帮助学生理解特征重要性与超参数的影响。

### 3. 训练可视：模型调参 (多层感知机动态透视)
* **功能说明**：动态展示神经网络的训练过程，洞悉梯度下降逻辑。
* **使用说明**：
  * 拖拽滑块调整模型的“学习率”和“训练轮数”。
  * 点击“开始动态训练”，系统将实时绘制 Loss 函数的下降曲线。
  * 学生可亲眼目睹曲线的平滑收敛与剧烈震荡现象，直观理解不同学习率导致的结果差异。

### 4. 智能答疑：助教中心 (大模型云端赋能)
* **功能说明**：全天候在线的农业 AI 专属虚拟助教，结合农业语境提供精准解答。
* **使用说明**：
  * 在下方输入框输入遇到的人工智能算法疑问或代码报错信息。
  * 内置的国产先进大模型 (如 DeepSeek API) 会自动结合上下文，以通俗易懂的方式给出解答与指导建议，极大降低试错挫败感。

### 5. 交互编程：农智沙箱 (云端伪沙箱环境)
* **功能说明**：免除配置 Python 与 PyTorch 等环境的折磨，提供开箱即用的轻量级交互编程体验。
* **使用说明**：
  * 选择预置的经典农业 AI 代码模板（如：产量预测线性回归、小麦病害 CNN 搭建、物联网数据清洗）。
  * 在代码编辑器中自由修改代码。
  * 点击“运行代码 (Run Cell)”，系统将安全的拦截代码执行结果并实时显示在下方的控制台面板中，做到知行合一。

### 6. 仿真实验：农机驾驶 (数字孪生寻路测试)
* **功能说明**：将启发式搜索理论（如 A* 算法）具象化为动态寻路与避障任务，打通 AI 落地的最后一公里。
* **使用说明**：
  * 拖动滑块调整虚拟农田环境中的“障碍物密度”。
  * 选择控制算法。
  * 点击“启动自动驾驶仿真”，观察虚拟拖拉机/无人机在随机生成的复杂农田中进行全局最优路径规划及自动避障的完整过程。

## 🚀 运行与部署说明

本项目采用单机轻量化架构，所有界面均由现代化 GUI 库 `CustomTkinter` 渲染，数学计算与绘图依赖开源的 `Numpy`, `Scipy`, `Scikit-learn` 及 `Matplotlib`。

### 源码运行方式 (适合开发者)
1. **环境准备**：
   请确保您的电脑已安装 Python 3.8 或以上版本。
2. **克隆/下载项目**：
   将本项目代码下载到本地。
3. **安装依赖库**：
   在项目根目录下打开终端，执行以下命令安装必要的依赖：
   ```bash
   pip install customtkinter numpy matplotlib scikit-learn scipy pillow requests manim
   ```
4. **启动系统**：
   在终端中运行主程序：
   ```bash
   python main.py
   ```
### 源码运行方式（适合教师）
1. **下载**
   下载页面右侧Release中的可执行文件Agri-AI-Explorer.exe。
2. **运行**
   双击运行Agri-AI-Explorer.exe即可。
3. **等待**
   由于机器学习torch包较大，运行后请耐心等待几分钟，直到程序启动。

### 大模型 API 配置指南
为了让“智能答疑”模块连接到真实的云端 AI 助教，您需要在代码中填入真实的 API Key：
1. 使用文本编辑器打开 `main.py`。
2. 找到 `call_llm_api` 函数（约在 600 行左右）。
3. 将 `API_KEY = "sk-..."` 替换为您申请的国产大模型 API 密钥（如 DeepSeek 等兼容 OpenAI 接口的模型）。

### 打包发布 (绿色免安装版)
项目已内置 PyInstaller 打包配置 `Agri-AI-Explorer.spec`，可通过以下命令将其打包为一个可直接在 Windows 系统上运行的单一 `.exe` 文件：
```bash
pyinstaller Agri-AI-Explorer.spec
```
打包完成后，将 `dist` 目录下的 `Agri-AI-Explorer.exe` 及相关 `assets` 资源分发给学生，学生无需安装任何环境，双击即可直接使用！
